# Timer & Stopwatch - ESP32 + 16x2 LCD

A small device built using an ESP32 microcontroller, buttons, and a 16x2 LCD. 

[Demo](https://www.youtube.com/shorts/_BMoXIRkDfw) (Link to a YouTube Short)

## Features
- **Timer Mode** - choose from 30 min, 45 min, or 1 hour presets
- **Stopwatch mode** - caps at 9h 59m 59s
- Pause, resume, reset and end functionalities
- Live time display
